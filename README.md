# ChargeNow

> Chargebee is a subscription management system which can help you handle all the aspects of the subscription life cycle including recurring billing, invoicing and trial management for your customers.

## Built With

- NextJS
- NodeJS
- ExpressJS
- MongoDB
- Socket.Io
- Nodemailer
- Chakra UI
